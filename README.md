# 🚀 AWS Certification Quiz Simulator

Un simulateur d'examen interactif basé sur une banque de 710 questions réelles. Ce projet permet de s'entraîner dans les conditions réelles de l'examen (65 questions, temps limité, calcul du score à l'échelle).

## 🛠️ Fonctionnalités

- **Tirage Aléatoire :** Sélection de 65 questions parmi 710 à chaque session.
- **Conditions Réelles :** - Limite de temps de **130 minutes**.
    - Gestion des questions **scored (50)** et **unscored (15)** comme à l'examen officiel.
    - Score final calculé sur une échelle de **1000 points** (Seuil de réussite : 720).
- **Rapport de Performance :** Correction détaillée à la fin du quiz affichant les bonnes et mauvaises réponses.
- **Interface Intuitive :** - Blocage de la navigation si aucune réponse n'est sélectionnée.
    - Fenêtre de confirmation personnalisée avant la soumission finale.

## 💻 Stack Technique

- **Frontend :** HTML5, CSS3 (Flexbox/Grid), JavaScript Vanille.
- **Backend (Parsing) :** Node.js pour la conversion des données brutes .txt vers JSON.
- **Données :** Format JSON structuré pour une gestion rapide des questions à choix unique et multiple.
